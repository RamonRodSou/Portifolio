Sobre mim

Olá, me chamo Ramon Rodrigues e seja bem-vindo ao meu portfólio.

Vou compartilhar um pouco sobre mim:

Iniciei na programação estudando lógica de programação com a tecnologia JAVA. Foi um passo ousado, mas desde então, tenho vivido uma incrível jornada de aprendizado e crescimento.

  • Python: Estou me aventurando na análise de dados com Python, automação de tarefas e aplicativos mobile, explorando seu vasto potencial.
  • Graduação em Análise e Desenvolvimento de Sistemas: Estou comprometido em expandir meu conhecimento acadêmico, e atualmente estou cursando Análise e Desenvolvimento de Sistemas na Unicesumar.

  • AluraGeek

    Projeto

    O projeto AluraGeek relizado utilizando HTML, CSS e JavaScript é o resultado de um desafio empolgante que surgiu durante minha participação no programa Oracle Next Education, enquanto estava estudando pela plataforma de cursos online Alura. Esse desafio consistia em criar um site de compras que utilizasse operações CRUD (Create, Read, Update, Delete) para gerenciar os produtos disponíveis.

    No entanto, eu não me limitei apenas a isso. Fui além e desenvolvi funcionalidades adicionais que tornam o AluraGeek uma plataforma completa. Além do gerenciamento de produtos, implementei um sistema de cadastro de usuários, com distinção entre usuários normais e administradores. Essa diferenciação é fundamental, pois apenas os administradores têm permissão para criar, deletar, atualizar novos produtos e editar ou excluir informações dos clientes.

    Com o AluraGeek, tenho orgulho de oferecer uma experiência de compra online abrangente e segura. Os administradores têm controle total sobre o catálogo de produtos, garantindo que os clientes tenham acesso a uma ampla variedade de opções. Ao mesmo tempo, os clientes podem criar contas, realizar compras e desfrutar de um ambiente amigável e funcional.

    Estou entusiasmado em compartilhar o AluraGeek como um exemplo de como a educação e o aprendizado contínuo podem inspirar soluções criativas e práticas para desafios do mundo real. Este projeto reflete meu compromisso em aplicar o conhecimento adquirido por meio da educação online para criar soluções significativas e inovadoras.

  • Organo

    Projeto

    Este projeto é um exemplo empolgante do que pode ser alcançado com a tecnologia React JS. Foi desenvolvido durante um curso ministrado pela Alura, que proporcionou uma experiência de aprendizado prática e enriquecedora.

    A ideia por trás deste projeto é a gestão de colaboradores e times de forma eficiente e intuitiva. Cada colaborador pode ser cadastrado com informações como nome, cargo e uma foto pessoal. Além disso, cada colaborador é associado a um time específico. A singularidade deste projeto reside na possibilidade de personalizar cada time com uma cor distinta.

    No momento do cadastro do time, os usuários têm a liberdade de escolher a cor que melhor representa o seu grupo. Isso não apenas torna a identificação visual mais fácil, mas também adiciona um toque de personalização que torna o projeto mais envolvente.

    Ao utilizar React JS, conseguimos criar uma experiência de usuário suave e responsiva. Os conceitos e técnicas aprendidos durante o curso da Alura permitiram que este projeto se tornasse uma aplicação prática e funcional para a gestão de colaboradores e times.
